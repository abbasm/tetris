tetris
======

### A HTML5 canvas simple tetris lib 

DEMO : http://tetrislib.herokuapp.com

To create a Tetris object:

    var tetris = new Tetris(element);
        element => DOM element in which the tetris object should be placed
  
Start/Stop :

    tetris.stop()

Restart:

    tetris.restart()

Cursol key :

    tetris.keyPress(key)
